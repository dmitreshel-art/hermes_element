## Что уже зафиксировано локально
- сервер: `matrix.22brn.ru`
- вход: логин / пароль
- VPN для обычного входа не требуется
- отдельный 2FA для текущего сценария не используется
- BYOD разрешён для личных телефонов и ноутбуков
- внешних участников без отдельного подтверждения приглашать нельзя
- основная точка обращения по поддержке и безопасности: **напишите этому боту**

## Правило качества
Если факт неизвестен локально, агент не должен его выдумывать.

## Редакторское правило
Даже если формальный локальный регламент ещё не дописан, в user-facing KB не должно оставаться сырых служебных маркеров и шаблонных заглушек. Вместо этого пишите либо подтверждённый локальный факт, либо честную временную формулировку вроде «точный лимит сейчас не зафиксирован, уточните через этого бота».
