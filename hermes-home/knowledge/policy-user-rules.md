# Правила использования служебного мессенджера

## Для чего используется служебный мессенджер
- Для рабочих коммуникаций, согласований и переписки по задачам.
- Для внутренних комнат и пространств, связанных с работой.

## Базовые правила
- Перед отправкой сообщения или файла проверяйте, что выбрана правильная комната.
- Не отправляйте чувствительные данные без необходимости.
- Если сомневаетесь, подходит ли комната для конкретной информации, сначала уточните через этого бота.

## Комнаты и пространства
- Используйте понятные названия рабочих комнат.
- Если создаёте новую внутреннюю комнату, выбирайте название, по которому другим сотрудникам будет ясно назначение комнаты.
- Внешних участников и подрядчиков без отдельного подтверждённого решения приглашать не нужно.

## Упоминания
- Личные упоминания используйте, когда нужен ответ конкретного человека.
- Массовые упоминания используйте только для действительно важных сообщений.

## Файлы и ссылки
- Перед отправкой файла проверьте адресата и комнату.
- Если файл содержит чувствительную информацию и вы не уверены, что его можно отправлять, лучше сначала уточнить через этого бота.

## Что делать при ошибке
- Если сообщение или файл ушли не туда и там есть чувствительные данные, сразу напишите этому боту как о вопросе безопасности.
- Если видите незнакомую сессию или предупреждение про ключи/сброс Secure Backup, не действуйте наугад и тоже сразу пишите этому боту.
