# Как подтвердить новое устройство

## Важно
Подтверждение устройства связано с доступом к зашифрованной истории и ключам. Не выполняйте рискованные действия, если не понимаете последствия.

## Безопасный общий порядок
1. Войдите в Element на новом устройстве.
2. Дождитесь запроса на подтверждение устройства.
3. Если старое доверенное устройство доступно, подтвердите новое устройство через него.
4. Если старое устройство недоступно, используйте recovery key / Secure Backup только если он у вас уже есть и вы понимаете, что вводите его локально в интерфейсе Element.
5. После подтверждения проверьте доступ к зашифрованным сообщениям.
6. Если старая история не открывается, не нажимайте reset наугад — сначала напишите этому боту.

## Чего не делать без инструкции
- не сбрасывать ключи шифрования наугад;
- не нажимать reset secure backup / reset identity только чтобы убрать предупреждение;
- не выходить со всех устройств без понимания последствий;
- не удалять старую доверенную сессию, если через неё ещё можно подтвердить новое устройство;
- не отправлять recovery key, пароль, security phrase, access token или коды подтверждения в чат.

## Если не помогло
Напишите этому боту и передайте:
- тип устройства;
- ОС и версию клиента;
- текст ошибки или скриншот;
- есть ли доступ к старому устройству;
- есть ли recovery key / Secure Backup, если он был ранее настроен;
- читается ли старая зашифрованная история на других устройствах.
