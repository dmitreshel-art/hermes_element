# Стартовый FAQ для помощника по Element

## 1. Как войти в Element впервые?

- Используйте рабочий клиент Element для сервера `matrix.22brn.ru`.
- Вход выполняется по обычным логину и паролю.
- VPN для обычного входа не требуется.
- Отдельный второй фактор в этом сценарии не используется.
- После входа проверьте имя, список комнат и уведомления.

Если вход не получается:
- проверьте логин/пароль и раскладку клавиатуры;
- попробуйте другой клиент: web / desktop / mobile;
- напишите этому боту и приложите текст ошибки.

## 2. Как настроить уведомления?

- Откройте настройки клиента.
- Перейдите в раздел уведомлений.
- Для обычной работы используйте режим, при котором важные комнаты не заглушены, а лишние массовые уведомления сведены к минимуму.
- Для отдельных комнат настраивайте уведомления отдельно.

## 3. Что делать при новом устройстве?

- Войдите в учётную запись.
- Если система предлагает подтвердить устройство, используйте уже доверенное старое устройство или recovery key / Secure Backup, если он у вас есть.
- Не сбрасывайте ключи шифрования, пока не проверите наличие recovery key или другой активной сессии.
- Если не уверены, что делать, сначала напишите этому боту.

## 4. Что делать, если не видно старую переписку?

Возможная причина — отсутствуют ключи шифрования или новое устройство ещё не подтверждено.

Что сделать:
- проверить, есть ли другая активная сессия;
- проверить наличие recovery key;
- подтвердить новое устройство;
- если доступ не восстанавливается, написать этому боту.

## 5. Какие правила использования служебного мессенджера важно помнить?

- Используйте служебный мессенджер для рабочих коммуникаций.
- Не отправляйте чувствительные данные без необходимости и без уверенности, что выбрана правильная комната.
- Внешних участников без отдельного разрешения приглашать не нужно.
- При подозрении на компрометацию сразу пишите этому боту.

## 6. Когда нужно обращаться в ИТ, а не пытаться решать самому?

Сразу пишите этому боту, если:
- не удаётся войти после стандартных шагов;
- есть проблемы с ключами шифрования;
- есть подозрение на взлом учётной записи;
- нужна помощь с правами, комнатами или служебными пространствами.
