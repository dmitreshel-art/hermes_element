# Стартовый FAQ для помощника по Element

> Это стартовая локальная памятка. Её можно дальше уточнять по мере накопления практики поддержки.

## 1. Как войти в Element впервые?

- Откройте `https://matrix.22brn.ru`.
- Нажмите "Войти".
- Используйте вашу учётную запись и вход по логину и паролю.
- Второй фактор не используется.
- После входа проверьте имя, аватар и уведомления.

Если вход не получается:
- VPN не требуется;
- проверьте, что используете `Element Web, Desktop, Android, iPhone`;
- напишите этому боту.

## 2. Как настроить уведомления?

- Откройте настройки клиента.
- Перейдите в раздел уведомлений.
- Рекомендуемый режим для сотрудников: `все сообщения`.
- Для важных комнат используйте персональные настройки комнаты.

## 3. Что делать при новом устройстве?

- Войдите в учётную запись.
- Если система предлагает подтвердить устройство — используйте процедуру `через старое устройство`.
- Не сбрасывайте ключи шифрования, пока не проверите наличие recovery key / другой активной сессии.
- Если старые сообщения не расшифровываются, напишите этому боту. Подробная памятка по recovery key и Secure Backup уже добавлена в локальную справку.

## 4. Что делать, если не видно старую переписку?

Возможная причина — отсутствуют ключи шифрования или новое устройство ещё не подтверждено.

Что сделать:
- проверить, есть ли другая активная сессия;
- проверить наличие recovery key;
- открыть локальную памятку по recovery key и Secure Backup;
- если доступ не восстанавливается, напишите этому боту.

## 5. Какие правила использования служебного мессенджера важно помнить?

- Используйте служебный мессенджер только для рабочих коммуникаций в рамках политики компании.
- Не публикуйте чувствительные данные вне разрешённых комнат.
- Соблюдайте правила именования и назначения комнат.
- При подозрении на компрометацию сразу сообщайте в поддержку.

## 6. Когда нужно обращаться в ИТ, а не пытаться решать самому?

Сразу обращайтесь в ИТ, если:
- не удаётся войти после стандартных шагов;
- есть ошибка входа, которую не удаётся устранить обычной проверкой логина и пароля;
- есть проблемы с ключами шифрования;
- есть подозрение на взлом учётной записи;
- нужна настройка прав, комнат или служебных пространств.
