# Эскалация в ИТ и ИБ

## Куда писать
- Основная точка обращения и по обычным вопросам, и по вопросам безопасности — **напишите этому боту**.
- Если случай требует участия специалиста, обращение будет дальше передано по нужному каналу.

## Когда обращаться по обычной поддержке
Пишите этому боту, если:
- не удаётся войти после штатных шагов;
- не приходят уведомления после базовой проверки;
- не удаётся загрузить файл или использовать обычную функцию клиента;
- нужна помощь по комнатам, доступам или настройкам клиента.

## Когда это уже вопрос ИБ / инцидента
Тоже пишите этому боту, но сразу указывайте, что это похоже на инцидент безопасности, если:
- есть подозрение на взлом аккаунта;
- обнаружена незнакомая сессия;
- чувствительные данные отправлены не туда;
- есть признаки компрометации устройства;
- Element предлагает сброс Secure Backup / identity, а вы не понимаете последствия;
- пропал доступ к зашифрованной истории после рискованного действия.

## Что приложить к обращению
- краткое описание проблемы;
- время возникновения;
- тип устройства и ОС;
- версия клиента;
- текст ошибки;
- скриншот, если это уместно;
- для проблем со входом — в каком клиенте это произошло: web / desktop / mobile.

## Практическое правило
- Если это обычная рабочая проблема с клиентом — просто напишите этому боту.
- Если есть риск для аккаунта, устройств, ключей или переписки — тоже напишите этому боту, но сразу отметьте, что это вопрос безопасности.
